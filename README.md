# Affirmation AI

Personalized AI Affirmation Generator built with a focus on simplicity and function. Users input a goal or fear, and the AI generates tailored affirmations. Requires a one-time purchase for instant access.

**Features:**
- Goal/fear input
- AI-generated affirmations (mocked in demo, real integration with OpenAI API)
- Simple UI
- Placeholder for Stripe Checkout integration

**To Run (Development Mode):**
1. Clone this repository.
2. Open `index.html` in your browser.

*(Note: AI generation and Stripe integration are placeholders in this front-end only demo. A full implementation would require a backend.)*
